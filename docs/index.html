Yancy is an API spam detector powered by magic.

You can register for an API key by following @kameronkales on Twitter.

I have not yet set up a sign up flow.

The free tier can use up to 50 requests a month.

I set this limit so I don't wake up with a $300 AWS bill. If you would like to use the service more please let me know.

Once you have an API key using the service is very easy.

You send 1 POST request to https://api.yancy.app/spam with the following in the body:

{"api_key": "insert your api key here", "spam": "insert your content here"}

You will receive a 200 Response if nothing breaks matching the format below:

{ "response": 200, "results": "False" }

No links are allowed and most sexual comments are flagged as spam.

The current accuracy score in my tests is 91%-ish.

The model retrains nighly.
